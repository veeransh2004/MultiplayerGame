# Multiplayer Lag Simulation

## Prerequisites
* Java Development Kit (JDK) installed
* Git installed

## Instructions

### 1. Clone the Project
Open your terminal and run:
```bash
git clone https://github.com/veeransh2004/MultiplayerGame.git
```
### 2. Complie 
Run the following bash commands to complie to GameServer.java and GameClien.java
```bash
javac GameServer.java GameClient.java
```
### 3. Start the Server, run the following command in terminal
```bash
java GameServer
```
### 4. Open a new terminal and run
```bash
java GameClient
```
To add multiple players we can create a new Tab and repeat step 4
